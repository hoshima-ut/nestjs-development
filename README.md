# nestjs-development

[nestjs](https://nestjs.com/)開発勉強用リポジトリです
## Doc

### 1. start

```bash
$ sh start.sh
```
### 2. Running the app


```bash
$ docker compose exec app ash
# watch mode
$ npm run start:dev
```

http://localhost:12345/

[read more](https://github.com/hoshima-ut/nestjs-development/tree/main/app)

### 3. stop

```bash
$ sh stop.sh
```
